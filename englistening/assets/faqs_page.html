What is this app?

The English Listening app was created to help you improve your English listening.

● 6 different types of listening lessons

● 30 lessons in each type

● 180 lessons in the entire program

The lessons were created for both beginners and advanced English learners. By studying this app, you will improve your English listening and improve your listening test scores.

How do the points work?

Easy lessons are given less points, while harder lessons are given more points. For example, Lesson #1 is only worth 5 points, while Lesson #30 is worth 20 points.

Also, the points are less if you listen to the audio file many times.

Lesson #1 example:

Listen to audio file once: 5 points

Listen to audio file twice: 3 points

Listen to audio file many times: 1 point

Lesson #30 example:

Listen to audio file once: 20 points

Listen to audio file twice: 15 points

Listen to audio file many times: 7 point

The points next to the lesson are the number of points for that lesson.

The number below each category is the total number of points for all the lessons in that category.

The “View the text” button does not work!

If you view the text, then you can answer all the questions correctly, so we do not turn on the button in the beginning.

Is Internet connection required to use this app?

Yes. Internet connection is required to download the audio files.

We did our best to thoroughly test this app for all types of errors. However, we could have missed something so please email us at talkenglish@talkenglish.com with any errors that you see and we will fix it as soon as possible.
